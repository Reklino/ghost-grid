Blooby
Grid

An itty bitty tool for visualizing fluid grids during development.
Scroll Down to see how to use.

Basic Fluid Grid

The container property sets the maximum width of the grid. It should be set to the width of your content. 


bloobyGrid.container = 960,
bloobyGrid.containerPosition = 'center',
bloobyGrid.breaks = [
    {
        point : "(min-width: 1200px)",
        columns : 6,
        gutters : .166,
        baseLineHeight : 24
    }
];
bloobyGrid.gridInit();

Full Width Container

The container property sets the maximum width of the grid. It should be set to the width of your content. 


bloobyGrid.container = null,
bloobyGrid.breaks = [
    {
        point : "(min-width: 900px)",
        columns : 12,
        gutters : .33,
        baseLineHeight : 36
    }
];
bloobyGrid.gridInit();
